# 🚀 TaskFlow AI

AI-powered real-time collaborative project management platform.

## Features

- JWT Authentication
- Kanban Board
- Drag & Drop
- MongoDB
- Socket.io
- AI Assistant
- Team Collaboration
- Responsive Design

## Tech Stack

- React
- Vite
- Tailwind CSS
- Node.js
- Express
- MongoDB
- Socket.io
- Gemini AI

## Installation

### Client

```bash
npm install
npm run dev
```

### Server

```bash
npm install
npm run dev
```
